# Donation Dashboard (Enhanced)

A small React + Vite + Tailwind CSS project that showcases a Donation Tracking Dashboard.
Includes charts (Recharts), CSV export, month-range filter, and polished UI.

## Quick start

1. Install dependencies
```bash
npm install
```

2. Run dev server
```bash
npm run dev
```

3. Build for production
```bash
npm run build
```

4. Preview production build
```bash
npm run preview
```

## Deploy

You can deploy to Vercel or Netlify. For both, connect your GitHub repo and use the default build command `npm run build` and output directory `dist`.
